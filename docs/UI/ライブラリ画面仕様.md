# ライブラリ画面仕様

## 画面構成

### レイアウト
- **2カラムレイアウト**
  - 左サイドバー: ライブラリ一覧（320px）
  - メインエリア: ライブラリ詳細・ファイル管理

### レスポンシブ対応
- タブレット（768px以下）: タブ切り替え方式
- モバイル（640px以下）: ドロワーナビゲーション
