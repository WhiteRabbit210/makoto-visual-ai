# タスク管理画面仕様

## 画面構成

### レイアウト
- **カンバンボード形式**
  - 3列構成: 未着手(Pending)、進行中(In Progress)、完了(Completed)
  - ドラッグ&ドロップ対応
  - レスポンシブ対応（モバイルでは縦スクロール）
