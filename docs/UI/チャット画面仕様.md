# チャット画面仕様

## 画面構成

### レイアウト
- **3カラムレイアウト**
  - 左サイドバー: チャットリスト（280px）
  - メインエリア: チャット画面
  - 右サイドバー: PDFビューア（可変幅、デフォルト400px）

### レスポンシブ対応
- タブレット（768px以下）: サイドバーをドロワー化
- モバイル（640px以下）: シングルカラム表示

## 状態管理

### ローディング状態
- メッセージ送信中: 送信ボタンをローディングアイコンに変更
- ストリーミング中: カーソル表示とスケルトン
- ファイルアップロード中: プログレスバー表示

### エラー状態
- ネットワークエラー: トースト通知
- API制限: エラーメッセージ表示
- ファイルサイズ超過: インラインエラー

### 空状態
- チャットリストが空: 「新しいチャットを始める」プロンプト
- メッセージが空: ウェルカムメッセージ表示
